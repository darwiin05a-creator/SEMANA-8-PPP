El comando **uname -a** muestra información del sistema operativo, como el nombre del sistema, la versión del kernel y la arquitectura del equipo.

El comando **whoami** muestra el nombre del usuario que está utilizando el sistema.

Conocer esta información es útil para administrar un servidor, verificar el entorno de trabajo, solucionar problemas y comprobar que se tienen los permisos adecuados para realizar determinadas tareas.
